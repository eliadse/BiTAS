# BiTAS
Analisis pipeline to detect transient gamma-ray signals in any point of the field of view of observations from Imaging Atmospheric Cherenkov Telescopes (IACTs).

## Project history

This codebase has been under active development since 2018.
It was maintained as a private/local repository in the MAGIC collaboration data center and made public in 2025.
Earlier development is therefore not reflected in the GitHub commit history.
